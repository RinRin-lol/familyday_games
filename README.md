# FamilyDayゲームツクール特設サイト

ファミリーデイ向けに作成した、ゲーム作成支援サイトです。  
質問に答えるだけで、Google AI Studio に貼り付けるためのプロンプトを作れます。

## このサイトでできること

- 新しいゲームを作るためのプロンプト作成
- 作ったゲームを改良するためのプロンプト作成
- エラー相談用のプロンプト作成
- 生成したプロンプトのワンクリックコピー
- 低学年モード（ひらがな中心表示）への切り替え

## 主な特徴

- 子ども向けに大きめのUIと分かりやすい文言
- モード切替で入力フォームを最小化
- 「その他」選択時のみ自由入力欄を表示
- 低学年モードON時は、UI文言・選択肢・プレースホルダ・生成プロンプトをひらがな中心で表示
- iPad での利用を意識したプロンプト文面

## 画面の流れ

1. STEP 1 で質問に回答
2. STEP 2 にプロンプトを自動生成
3. コピーして Google AI Studio に貼り付け
4. 必要なら「改良する」「エラー相談」モードで再生成

## フォルダ構成

```text
familyday_games/
	index.html      # 画面構造
	styles.css      # 見た目
	app.js          # 入力制御・プロンプト生成
	gameLogic.js    # ゲームロジック補助
	arcade/         # 作ったゲームで遊ぶページ群
```

## 開き方

このプロジェクトはビルド不要の静的サイトです。

1. index.html をブラウザで開く
2. または VS Code の Live Server などで起動する

## 実装メモ

### モード

- make: 新しく作る
- improve: 改良する
- fix: エラー相談

### 低学年モード

- 画面上部の「低学年モード」ボタンで ON/OFF を切替
- ON の間は、できるだけひらがな中心の表示に切替
- 生成されるMarkdownプロンプトも、低学年向けの表現に切替

### 「その他」自由入力の仕様

- 対象の select に「その他」を用意
- 「その他」選択時のみ対応 input を表示
- 通常選択に戻したら自由入力値をクリア
- 生成時は選択値ではなく自由入力値を優先

対象項目:

- ゲームの種類
- 世界観
- 操作方法
- 難しさ
- 見た目の雰囲気
- 何を改良したい？
- こまっていること

## カスタマイズ方法

### 選択肢を増やす

1. index.html の該当 select に option を追加
2. 「その他」連動が必要なら app.js の otherSelects に追記

### プロンプト文章を調整する

- app.js の以下を編集
	- makeGamePrompt
	- makeImprovePrompt
	- makeFixPrompt
